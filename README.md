## Gmail API in Chrome Extension Demo

Simple Google Chrome extension that displays the signed in Chrome user's Gmail inbox. Originally was going to be a completely different extension but ran into technology limitations for what I wanted to do. 

If you are using this as a reference to do something similar, check out my other repo [Packages for Gmail Back-End](https://github.com/anatelli10/packages-for-gmail-back-end/blob/a82fea7978be26dd21e697dfa0bcafa10ddf90e9/accounts/account.service.js#L108) for implementations of message body parsing and retrieving more than 100 messages.
